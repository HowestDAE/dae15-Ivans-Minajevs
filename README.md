## Getting Started

### Prerequisites

*7Zip or WinRAR
* Visual Studio 2022

### How to run the project

* Unpack the project
* Open the sln file in Visual Studio Enterprise
* Set Dependencies to the Engine
* Set Game as default
* Run the project

<!-- HOW TO PLAY -->
## How to play

### Controls
* LEFT arrow - move left
* RIGHT arrow - move right
* DOWN arrow - move down
* UP arrow - enable throw

* Z - attack
* X - jump
